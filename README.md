# AI Interior Designer

Generate stunning AI-powered interior designs using natural language descriptions and Hugging Face FLUX.

## Features

- AI Image Generation
- 8 Interior Design Styles
- Prompt Engineering
- Prompt History
- Download Generated Images
- Random Room Inspiration
- Modern Dark UI

## Tech Stack

- Python
- Streamlit
- Hugging Face FLUX.1 Schnell
- Pillow
- Python Dotenv

## Setup

1. Clone repo
2. Install requirements

pip install -r requirements.txt

3. Create .env

HF_API_TOKEN=your_token_here

4. Run

streamlit run app.py
